# 訪問先編集セクション UI改善 — タイムライン強化型

**日付:** 2026-04-22  
**対象:** `index.html` サイドバー（`#sidebar`）内の訪問先リスト

---

## 背景・目的

訪問看護スタッフ（ITに不慣れな初心者を含む）が日々使うアプリ。現在のUIは機能的だが、初見では操作方法がわかりにくい箇所がある。4つの課題を解消し、直感的に使えるUIへ改善する。

---

## 改善箇所（4点）

### ① 訪問先挿入ボタンを大型化

**現在:** 各カード下部に `font-size: 11px` の小さなpillボタン2つ（「お気に入りから追加」「地図から追加」）  
**改善後:** 幅100%の青い点線ボックスボタン「＋ ここに訪問先を追加」1つに統合し、タップした後にお気に入り/地図の選択肢を展開

- 出発カードの直後、各訪問先カードの直後、最後の訪問先と帰社の間に常時表示
- スタイル: `background:#eff6ff; border: 1.5px dashed #93c5fd; border-radius: 10px; padding: 7px 12px;`
- クリック時の動作は現行と同じ（`setShowFavs(true)` or `setMapMode('stop')` を選択肢として表示）

### ② ボタンにテキストラベルを追加

**現在:** ☆（お気に入り）・×（削除）のアイコンのみ。意味が不明。  
**改善後:** ラベル付きのカラーボタン

| ボタン | 変更前 | 変更後 |
|-------|--------|--------|
| お気に入り | `☆`（28×28px アイコンのみ） | `★ お気に入り登録`（背景:#fffbeb, 文字:#92400e, 枠:#fde68a） |
| 削除 | `×`（28×28px アイコンのみ） | `✕ 削除`（背景:#fef2f2, 文字:#b91c1c, 枠:#fca5a5） |
| 地図（出発/帰社） | `📍` アイコンのみ | `📍 地図`（背景:#f0fdf4, 文字:#059669, 枠:#6ee7b7） |

### ③ ドラッグハンドルを枠付きボックスに変更

**現在:** 3本の薄いグレー線（`w-4 h-0.5 bg-slate-300`）。ほぼ見えない。  
**改善後:** `background:#f8fafc; border:1px solid #e2e8f0; border-radius:6px; padding:3px 4px;` のボックスで囲んだ3本線。「ここを掴む」と視覚的に伝わる。

### ④ 縦タイムライン線を追加

**現在:** 各カードが独立して並んでいる。経路の流れが見えない。  
**改善後:** カードリスト全体を `padding-left: 26px` のラッパーで囲み、`::before` 擬似要素で縦線を描画。

```css
.tl-wrap::before {
  content: '';
  position: absolute; left: 10px; top: 16px; bottom: 16px; width: 2px;
  background: linear-gradient(to bottom,
    #10b981 0%, #10b981 12%,      /* 出発：緑 */
    #3b82f6 12%, #3b82f6 52%,     /* 訪問先：青 */
    #f97316 52%, #f97316 80%,     /* 訪問先2：橙（件数に応じて変化） */
    #64748b 80%, #64748b 100%     /* 帰社：グレー */
  );
}
```

各ノード（出発・訪問先・帰社）の左端に `position:absolute` のカラー丸バッジ（22×22px）を配置。

---

## 変更しないもの

- 機能ロジック（ドラッグ並び替え・お気に入り・地図選択・距離計算）は一切変更しない
- サイドバーの幅・全体レイアウト・モバイルタブ対応は維持
- 検索バー・下部パネル（出退勤時刻・合計距離）は変更しない
- `stop-item` クラス名はそのまま維持（ドラッグ機能と紐付いているため）

---

## 対象ファイル

- `index.html`（単一ファイル構成）
  - `<style>` セクション: `.stop-item`、新規 `.tl-wrap` などを追加
  - JSX部分: `rec.stops.map(...)` 周辺のマークアップを変更

---

## 成功基準

- 挿入ボタンが目立ち、初見で「ここを押せば追加できる」とわかる
- ☆ × ボタンの意味がラベルで明確になる
- ドラッグハンドルが視覚的に認識できる
- 出発→訪問先→帰社の縦の流れが線で見える
- 既存の機能（ドラッグ並び替え・距離表示など）が引き続き動作する
